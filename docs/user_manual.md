## 1. 安装插件

### 前置条件

- Blender 3.6 LTS 或更新版本
- **mmd_tools** 插件已安装并启用

### 安装步骤

1. 将 `RGBA_mmd/` 文件夹压缩为 `RGBA_mmd.zip`
2. 打开 Blender
3. 菜单：**编辑 → 偏好设置 → 插件**
4. 点击右上角 **安装（Install）**
5. 选择 `RGBA_mmd.zip`，点击 **安装插件**
6. 在插件列表中找到 **RGBA-Style MMD Bust Rig**，勾选启用

---

## 2. 导入模型

1. 菜单：**文件 → 导入 → MikuMikuDance Model (.pmx)**
2. 浏览到你的 PMX 文件，选中
3. 在右侧导入选项中确认勾选了 `Mesh`、`Armature`、`Physics`
4. 点击 **导入 MMD 模型**
5. 等待导入完成，场景中出现模型

> **提示**：如果场景中已有其他物体，建议先按 **A** 全选 → **Delete** 清空场景再导入。

---

## 3. 导入动作

1. 在 3D 视图中**点击选中模型的骨架**（Armature）
   - 可以在大纲视图（Outliner）中点击带有骨骼图标的对象
   - 或者直接点击模型的骨骼
2. 菜单：**文件 → 导入 → MikuMikuDance Motion (.vmd)**
3. 选择你的 VMD 文件
4. 点击 **导入动作**

> **注意**：必须先选中骨架再导入 VMD，否则动作无法正确加载。

---

## 4. 打开面板

1. 将鼠标移到 3D 视图区域
2. 按 **N** 键打开侧面板
3. 在侧面板顶部找到 **RGBA MMD** 标签，点击切换

面板分为四个区域：
- **状态栏**：显示上一次操作的结果
- **Blender 物理（刚体方式）**：传统刚体/关节方式
- **弹簧模拟（推荐）**：数学模拟方式
- **导出 PMX**：导出带物理的模型文件

---

## 5. 检测胸部骨骼

1. **选中模型中的任意物体**（网格、骨骼、根 Empty 均可）
2. 在面板中点击 **「检测胸部骨骼」**
3. 查看状态栏显示的信息

正常结果示例：
```
Model='模型名' arm='骨架名' parent=上半身2 bust=[boob left 1(L), boob right 1(R)]
```

如果显示 "No bust bones found"：
- 检查模型的胸部骨骼命名是否包含：`胸`、`乳`、`bust`、`breast`、`boob` 等关键字
- 如果命名不同，需要手动重命名或修改代码

如果显示 "No parent bone found"：
- 在面板下方 **刚体参数 → 骨骼 → Parent Bone** 中手动输入父骨骼名称

## 6. 方式一：弹簧模拟（推荐）

这是在 Blender 中获得可见弹跳效果的推荐方式。

### 操作步骤

1. 确认已完成：导入模型 → 导入动作 → 检测骨骼
2. 设置弹簧参数（或保持默认值）：
   - **Spring K**: 80（弹簧刚度）
   - **Damping**: 6（阻尼）
   - **Mass**: 1.0（质量）
   - **Scale**: 3.0（放大倍数）
3. 点击 **「运行弹簧模拟」**
4. 等待状态栏显示完成信息
5. 按 **Alt+A** 播放动画查看效果

### 调整效果

如果效果不满意：
1. 先点击 **「清除模拟关键帧」** 移除当前效果
2. 调整参数
3. 重新点击 **「运行弹簧模拟」**

### 参数快速参考

| 想要的效果 | Spring K | Damping | Mass | Scale |
|-----------|----------|---------|------|-------|
| 慢摆荡 | 40~60 | 6~8 | 1.5~2.0 | 2~3 |
| 蓬松弹 | 80~100 | 3~4 | 1.0 | 2~3 |
| 写实 | 100~150 | 10~15 | 1.0 | 1.5~2.0 |
| 夸张 | 60~80 | 4~6 | 1.0 | 4~5 |

> **原理**：弹簧模拟读取父骨骼（上半身2）每帧的旋转，通过弹簧-阻尼器公式计算出胸部骨骼的旋转偏移，直接写入关键帧。这绕过了 Blender 物理系统的限制。

## 7. 方式二：Blender 物理刚体

传统的 RGBA 式 5 刚体 + 8 关节方式。刚体/关节会保存到 PMX 文件中。

### 操作步骤

1. 确认已检测到胸部骨骼
2. （可选）展开 **「刚体参数」** 面板调整参数
3. 点击 **「应用 RGBA 刚体」**
4. 状态栏显示创建的对象数量

### 注意事项

- 此方式在 Blender 视图中**可能看不到明显弹跳**（因为 Blender Bullet 的零限制技巧不适用）
- 但导出为 PMX 后，在 **MMD / PMXEditor** 中可以正常工作
- 如果想在 Blender 中预览效果，请配合使用弹簧模拟（方式一）

### 移除刚体

点击 **「移除 RGBA 刚体」** 删除本插件创建的所有刚体和关节（不影响模型原有物理）。

---

## 8. 导出带物理的 PMX

将当前模型（含 RGBA 刚体/关节）导出为 PMX 文件。

### 前提

- 已执行 **「应用 RGBA 刚体」**（方式二）
- PMX 格式保存的是模型结构（刚体/关节），不保存动画关键帧

### 操作步骤

1. 在 **「导出 PMX」** 区域的路径栏中设置输出路径
   - 例如：`E:\mywork\mymodel\output.pmx`
   - 必须以 `.pmx` 结尾
2. 点击 **「导出带物理的 PMX」**
3. 状态栏显示导出结果（文件大小、刚体数、关节数）

### 导出后验证

- **PMXEditor**：打开导出的 PMX → 剛体标签页 → 应看到 11 个刚体
- **MMD**：加载模型 + VMD 动作 → 播放 → 胸部应有弹跳效果

---

## 9. 显示/隐藏刚体

点击 **「显示/隐藏刚体」** 切换所有刚体和关节对象在 3D 视图中的可见性。

- 显示状态下可以看到刚体的球体/关节的十字线
- 隐藏后恢复干净的模型视图
- 每次点击在显示/隐藏之间切换

> **提示**：也可以在 Outliner（大纲视图）中找到 `rigidbodies` 和 `joints` 集合，点击眼睛图标切换。

---

## 10. 参数调节指南

### 弹簧模拟参数

| 参数 | 默认值 | 范围 | 说明 |
|------|--------|------|------|
| **Spring K** | 80 | 1~500 | 弹簧刚度。越低弹簧越松，摆幅越大；越高越紧，跟随越快 |
| **Damping** | 6 | 0.1~50 | 阻尼系数。越低振荡持续越久；越高振荡越快停止 |
| **Mass** | 1 | 0.1~10 | 质量。越大惯性越大，动作越迟钝；越小反应越灵敏 |
| **Scale** | 3 | 0.1~10 | 放大倍数。直接乘以模拟结果，最简单的效果调节方式 |
| **Parent Bone** | （空） | - | 驱动弹跳的父骨骼。留空自动检测（上半身2 → 上半身） |

### 刚体参数（展开面板查看）

| 参数 | 默认值 | 说明 |
|------|--------|------|
| **Body Radius** | 0.10 | 刚体球半径 |
| **Main Mass** | 0.5 | 主胸部刚体质量 |
| **Aux Mass** | 0.05 | 辅助刚体质量 |
| **Linear Damping** | 0.7 | 线性阻尼 |
| **Angular Damping** | 0.7 | 角度阻尼 |
| **Strong Spring k** | 300 | 锁定轴弹簧刚度 |
| **Loose Spring k** | 50 | 自由轴弹簧刚度 |
| **Spring Damping** | 0.4 | 关节弹簧阻尼 |
| **Collision Group** | 15 | MMD 碰撞组（1~16） |

## 11. 完整工作流程示例

### 场景：为 MMD 模型添加胸部弹跳，在 Blender 预览并导出 PMX

```
步骤 1: 安装插件
         ↓
步骤 2: 文件 → 导入 → PMX 模型
         ↓
步骤 3: 选中骨架 → 文件 → 导入 → VMD 动作
         ↓
步骤 4: 按 N 打开侧面板 → RGBA MMD 标签
         ↓
步骤 5: 点击「检测胸部骨骼」→ 确认状态正常
         ↓
步骤 6: 点击「运行弹簧模拟」→ Alt+A 播放预览
         ↓
  ┌─ 效果满意？
  │
  ├─ 是 → 步骤 7
  │
  └─ 否 → 点击「清除模拟关键帧」→ 调整参数 → 回到步骤 6
         ↓
步骤 7: 点击「应用 RGBA 刚体」（写入刚体/关节到模型）
         ↓
步骤 8: 设置导出路径 → 点击「导出带物理的 PMX」
         ↓
步骤 9: 在 PMXEditor 或 MMD 中验证
```

### 如果模型变形了怎么办？

1. 按 **Ctrl+Z** 多次撤销回到正常状态
2. 或者关闭文件不保存，重新打开
3. 或者重新导入原始 PMX（最可靠）
4. 弹簧模拟方式不会改变模型形状，只添加骨骼关键帧

## 12. 常见问题

### Q: 为什么 Blender 物理（刚体方式）看不到弹跳？

**A**: RGBA 原版方案依赖 MMD Bullet 的"零限制技巧"（关节限制设为 0 时求解器误差产生振荡）。Blender 的 Bullet 更精确，零限制不产生振荡。请使用弹簧模拟方式。

### Q: 弹簧模拟和刚体方式可以同时用吗？

**A**: 可以。弹簧模拟写入骨骼关键帧（Blender 预览用），刚体方式写入物理设置（PMX 导出用）。两者互不影响。

### Q: 导出 PMX 后弹簧模拟的效果会保留吗？

**A**: 不会。PMX 只保存模型结构（网格、骨骼、刚体、关节），不保存动画关键帧。弹簧模拟的效果需要重新导入后重新运行。但 RGBA 刚体/关节会保留在 PMX 中，在 MMD 里零限制技巧会自然生效。

### Q: 导出的 PMX 在 MMD 中有弹跳效果吗？

**A**: 有。需要先执行「应用 RGBA 刚体」再导出。MMD 的 Bullet 求解器支持零限制技巧，刚体/关节会正常工作。

### Q: 如何恢复模型到原始状态？

**A**: 
- 弹簧模拟：点击「清除模拟关键帧」
- 刚体方式：点击「移除 RGBA 刚体」
- 如果模型已变形：重新导入原始 PMX

### Q: 支持哪些骨骼命名？

**A**: 自动检测包含以下关键字的骨骼：`胸`、`乳`、`bust`、`breast`、`chest`、`boob`、`oppai`、`tit`。如果你的模型使用其他命名，需要重命名骨骼。

### Q: 可以调整弹跳的方向吗？

**A**: 当前弹簧模拟基于父骨骼的 X 轴（点头）和 Z 轴（摆动）旋转。如需自定义，可修改 `operators.py` 中 `RGBAMMD_OT_spring_sim` 的 `dx`/`dz` 赋值逻辑。
